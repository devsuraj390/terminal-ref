## Terminal Commands

- **`~`** : Check home directory.
- **`cd /`** : Navigate to root directory (/).
- **`man`** : Check the details on the command. For eg. `man pwd`.
- **`ls -a`** : See hidden files in a directory.
- **`mkdir dir_name`** : Creates a directory.
- **`touch file_name`** : Creates a file inside a directory if it does not exist.
- **`rmdir`**: Removes an empty directory.
- **`rm -rf`** : Removed directory with files recursively.
- **`rm file_name`** : Removes a file from the system. Not moved to trash.
